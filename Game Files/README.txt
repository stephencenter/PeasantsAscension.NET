Launching the game:
    1. Open the "Peasant's Ascension" folder
    2. Run the file named "Peasant's Ascension.exe" in this folder
    3. You're good to go!
----------------------------------------------------
How to increase font size:
    1. Right click the logo in the top left of the window
    2. Click "Properties"
    3. Select the Font tab
    4. Change "Size" to whatever value you want.
